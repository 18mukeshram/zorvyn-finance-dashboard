# zorvyn-finance-dashboard

This project is a clean and responsive finance dashboard interface built as part of a frontend developer assessment.

## Features
- Financial summary cards (Balance, Income, Expenses)
- Balance trend chart
- Expense category chart
- Transactions table with search, filter, and sorting
- Role-based UI (Viewer/Admin)
- Insights section
- Dark mode
- Responsive design

## Tech Stack
- React (Vite)
- Tailwind CSS
- Recharts
- Zustand

## Project Structure
Modular component-based architecture with centralized state management.

## How to Run
npm install
npm run dev
